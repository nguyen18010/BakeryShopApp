# Bakery Shop App

Ứng dụng quản lý bán hàng cho tiệm bánh, hỗ trợ đặt hàng, quản lý đơn và giao diện quản trị.

## 🛠 Công cụ sử dụng
- Java, SQLite, Android Studio
- Draw.io (UML, BPMN, ERD)

## 📊 Vai trò Business Analyst
- Phân tích yêu cầu kinh doanh và mô hình hóa quy trình bán hàng.
- Thiết kế sơ đồ ERD, UML (Use Case, Class, Sequence, Activity) và BPMN workflows.

## 📂 Cấu trúc repo
- `src/` : Source code
- `database/` : Script tạo database
- `diagrams/` : Sơ đồ UML/BPMN/ERD (.drawio + .png)
